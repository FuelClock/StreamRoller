How to get your oaut token ...


Use the links below to retrieve your oath token to paste into the fields. It is worth saving this off to avoid having to retrieve it every time you want to update your login credentials. The links below will generate a new oauth token so if you have other programs using the oath you will need to update those to the new oath after changing it.
Before clicking the links make sure your browser is logged in to twitch using the account you are getting the details for
1) Log into twitch with the account you want the token for (bot or user)

   Go to: https://twitchtokengenerator.com.
    Select bot account and then copy the field "ACCESS TOKEN" and add 'oauth:' to the start.
    i.e. if the token is ABC123 then the token will be oauth:ABC123
    Note: if you use the same account for both you may find some features might not work. ie you won't be able to address the openAI ChatGPT bot from chat as it will ignore its own messages
    (if you and the bot are the same account this means it will ignore you too :D)
