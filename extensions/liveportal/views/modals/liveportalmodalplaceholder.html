Twitter

On startup you can post a Tweet to let people know you are live. There three parts to the tweet.
  1. Main message
  2. Extra message
  3. Random fact (If enabled)
Streamroller will calculate the length of the tweet and do the following options in order depending on size
  1. If there is enough space, post "main message" + "Extra Message" + "Random fact"
  2. If 1. doesn't fit then try "Main message" + "Random Fact"
  3. If 2. doesn't fit then try "Main Message" + "Extra Message"
